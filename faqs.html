Frequently Asked Questions

What is the difference between OpenGapps and NikGApps?

  • NikGApps comes with prebuilt YouTube Vanced and Lawnchair launcher.
  • NikGApps packages are created from the apks present in my personal device.
  • NikGApps follows a different partition mounting mechanism.
  • NikGApps has it's own set of definition of packages defined keeping users feedback in mind.
  • NikGApps installs all its applications and whitelist permissions in /product partition whereas OpenGapps stores it in /system partition.
  • NikGApps has it's own implementation of install configuration (defined in nikgapps.config and debloater.config).

The installation failed?

It could be because of sevaral reasons

  • In case of "Busybox setup failed. Aborting..." error, Push the package to /sdcard and try flashing again.
  • Make sure the downloaded file is not corrupted.
  • Make sure zip file opens properly in File Explorer.
  • Try mounting system manually in recovery before flashing the zip
  • If you are on A/B device, make sure you follow the procedure (Flash the rom, twrp, reboot to recovery, flash rom, twrp, reboot to recovery, flash gapps).

How to use nikgapps.config?

  • Nikgapps.config is placed in /sdcard/NikGApps folder automatically after first install with all the packages enabled for installation.
  • If you want to configure the intallation with your very first installation, extract nikgapps.config from the gapps zip and place it to /sdcard/NikGapps folder.
  • Edit the file and set 0 against the package that you want to skip, save the file, flash the gapps.
  • For e.g. if you want to skip YouTube during the installation, the config should read YouTube=0.
  • All the lines starting # in config file are comments. Adding comments (#) in front of Package name will not skip it's installation. i.e. # YouTube=0 will not skip the intallation.
  • 'files' Package in nikgapps.config is not GoogleFiles. it is a package of set of important files for gapps installations. Never skip that

How to use debloater.config?

  • Debloater recognizes full path as well as folder names that you want to delete in priv-app or app folder.
  • (Recommended) You can Just enter the name of the folder you want to delete. For e.g. YouTube. This would remove the YouTube folder wherever it is located in /system/app, /system/priv-app, /system/product/app, /system/product/priv-app.
  • If you want to delete a specific file/folder you can enter full path. For e.g. /system/app/YouTube.
  • However, the above one wouldn't work in case of SAR device since the partition path you're looking for can be /system_root/system/app/YouTube or /system/system/app/YouTube(Use the Folder method).

Is Addon Package compatible with Android 9 and 10 both?

  • Bromite and Bromite webview doesn't work well with Android 10 yet. Rest all works!

Why are Pixel Launcher and Lawnchair Launcher missing in Q packages?

  • Pixel Launcher requires overlay changes which are not part of gapps yet.
  • Lawnchair is in alpha status, it will be added back once it is stable enough.
